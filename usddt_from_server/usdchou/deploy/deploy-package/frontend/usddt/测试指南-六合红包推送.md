
## 🐛 常见问题

### Q1: 控制台没有打印 "🧧 收到新六合红包"
**原因**: 监听器未正确注册  
**解决**: 检查 LiuHe.vue 的 import 语句和 onMounted 逻辑

### Q2: 打印了日志但页面没刷新
**原因**: loadRedPackets() 函数有问题  
**解决**: 检查该函数是否正确调用 API 并更新响应式数据

### Q3: 第一次可以，第二次不行
**原因**: 监听器重复注册或未清理  
**解决**: 确认 onUnmounted 中的清理逻辑正确执行

### Q4: 偶尔能收到，偶尔收不到
**原因**: Socket 连接不稳定  
**解决**: 检查网络状况，添加重连逻辑
