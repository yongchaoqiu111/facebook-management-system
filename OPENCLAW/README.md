# OpenClaw AI Assistant

一个强大的AI助手项目，用于自动化任务和智能交互。

## 功能特性

- 🤖 AI驱动的任务自动化
- 💬 多渠道消息集成
- 📅 日程管理和提醒
- 📁 文件处理和整理
- 📧 邮件自动回复
- 🎯 智能脚本执行

## 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
```

2. 安装依赖
```bash
npm install
```

3. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件配置 API 密钥
```

4. 启动应用
```bash
npm start
```

## 开发

```bash
npm run dev
```

## 测试

```bash
npm test
```
