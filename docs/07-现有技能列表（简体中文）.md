# 现有技能列表

本文档列出了项目中现有的所有技能及其功能描述。

## 1. Facebook 技能

### 1.1 mcp-facebook-login
- **功能**：Facebook 登录
- **描述**：使用 Playwright 浏览器自动化登录 Facebook，支持加载和保存 cookie
- **文件**：`skills/facebook/facebook-skills.ts`

### 1.2 mcp-facebook-post
- **功能**：Facebook 发帖
- **描述**：在 Facebook 上发布帖子，支持添加文本和图片
- **文件**：`skills/facebook/facebook-skills.ts`

### 1.3 mcp-facebook-interact
- **功能**：Facebook 互动
- **描述**：在 Facebook 上执行点赞、评论、分享等互动操作
- **文件**：`skills/facebook/facebook-skills.ts`

### 1.4 mcp-facebook-search
- **功能**：Facebook 搜索
- **描述**：在 Facebook 上搜索关键词，返回相关帖子
- **文件**：`skills/facebook/facebook-skills.ts`

### 1.5 mcp-weibo-facebook-join-groups
- **功能**：Facebook 加入小组
- **描述**：自动搜索并加入 Facebook 小组，支持关键词搜索和问答处理
- **文件**：`skills/facebook-join-groups.ts`

### 6.1 任务调度器

- **功能**：新闻搜索
- **描述**：提供网页界面，允许用户输入关键词搜索新闻

- **功能**：任务调度
- **描述**：提供网页界面，允许用户设置和管理定时任务
- **文件**：`public/scheduler.html`

## 技能调用关系

1. **每日发帖流程**：
   - 调用 LLM 获取新闻内容
   - 生成帖子文本
   - 选择图片
   - 调用 Facebook 发帖技能发布帖子

2. **热搜词探索流程**：
   - 读取关键词
   - 搜索相关新闻
   - 生成帖子内容
   - 保存到 tiezi 目录

3. **新闻蒸馏流程**：
   - 从多个来源获取新闻
   - 使用 LLM 进行蒸馏和总结
   - 生成最终新闻内容

## 注意事项

- 所有技能都需要正确的配置和依赖项才能正常运行
- Facebook 相关技能需要有效的 cookie 才能登录
- LLM 相关技能需要有效的 API 密钥
- 任务调度系统需要保持服务器运行才能执行定时任务
