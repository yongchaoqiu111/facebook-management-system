# Facebook 自动化工具安装指南

## 📦 项目介绍

这是一个自动化社交媒体运营工具，支持以下功能：
- Facebook 自动加入小组
- Facebook 小组帖子分析与评论
- 热搜词探索与互动

## 🛠️ 安装步骤

### 1. 安装 Node.js
- 下载地址：https://nodejs.org/
- 推荐版本：16.x 或更高版本
- 安装完成后验证：
  ```bash
  node -v
  npm -v
  ```

### 2. 安装项目依赖
```bash
# 安装 npm 依赖
npm install

# 安装 Playwright 浏览器
npx playwright install
```

### 3. 配置 API 密钥
在项目根目录创建以下文件：

#### 3.1 大模型 API Key
创建文件：`data/llm-api-key2.txt`
```
sk-sp-e81ff7e4ffd8476e9***********6df4
```

#### 3.2 Facebook Cookie
创建文件：`cookie/facebook.txt`
- 从浏览器中复制 Facebook 登录后的 cookie
- 格式：每行一个 cookie

## 🚀 启动服务

# 启动服务器
node server.js

## ⏰ 设置定时任务

### 添加任务步骤：
1. 在控制页面选择技能（如"Facebook 小组评论"）
2. 设置执行时间（24小时制，如：14:35）
3. 点击"添加任务"按钮

### 任务状态说明：
- **已调度**：等待执行
- **执行中**：正在运行
- **执行成功**：任务完成
- **执行失败**：任务出错

## 📝 定时任务特点

- **每天自动执行**：设置一次，每天在指定时间自动执行
- **服务器重启后自动恢复**：服务器重启时会自动加载所有任务
- **永久生效**：除非手动删除，否则任务会一直执行

## 🗑️ 删除任务

在任务列表中找到要删除的任务，点击"删除"按钮即可停止该任务的执行。

## 🔧 常见问题

### Q1: 浏览器无法启动
**解决方案**：确保已正确安装 Playwright 浏览器
```bash
npx playwright install
```

### Q2: Facebook 登录失败
**解决方案**：更新 `cookie/facebook.txt` 文件中的 cookie

### Q3: API 调用失败
**解决方案**：检查 `data/llm-api-key2.txt` 文件中的 API Key 是否正确

### Q4: 任务不执行
**解决方案**：
1. 检查服务器是否正常运行
2. 检查任务时间设置是否正确
3. 查看终端日志了解具体错误信息
