## 🚨 风险与应对

### 风险 1：后端接口未按时完成
**概率**：中  
**应对**：
- 使用 Mock 数据开发
- 先完成 UI 逻辑，后对接真实接口

### 风险 2：任务 A 延期影响任务 B
**概率**：低  
**应对**：
- 任务 B 可以先写函数框架
- 任务 A 完成后快速集成

### 风险 3：WSS 事件不触发
**概率**：中  
**应对**：
- 检查 Socket 连接状态
- 检查事件名是否匹配
- 查看浏览器控制台日志
