### 二、Socket事件对齐

#### 文档规范（v1.0）
- `joinGroup` - 加入群组
- `leaveGroup` - 离开群组
- `groupMessage` - 接收群聊消息
- `privateMessage` - 接收私聊消息
- `redPacketCreated` - 红包创建
- `redPacketOpened` - 红包被领取
- `redPacketFinished` - 红包领完
- `redPacketExpired` - 红包过期
- `redPacketExceeded` - 红包超过阈值

### 四、数据格式适配

#### 消息对象（API文档标准）
```javascript
{
  _id: string,
  sender: {
    _id: string,
    username: string,
    avatar?: string
  },
  groupId?: string,
  receiverId?: string,
  content: string,
  type: 'text' | 'image' | 'video' | 'file',
  clientMsgId?: string,
  createdAt: string  // ISO格式
}
```

#### 红包对象（API文档标准）
```javascript
{
  _id: string,
  sender: {...},
  chainGroupId?: string,
  totalAmount: number,
  count: number,
  remainCount: number,
  remainAmount: number,
  totalClaimed: number,
  status: 'active' | 'finished' | 'expired' | 'exceeded',
  isChainRedPacket: boolean,
  claims: [{ userId, username, amount, claimedAt }],
  expiredAt?: string,
  createdAt: string
}
```
