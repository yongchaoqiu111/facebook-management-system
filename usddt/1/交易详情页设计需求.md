# 交易详情页面设计需求

## 📄 页面名称
交易详情页 (TradeDetail.vue)

## 🎯 页面功能
展示单个交易对的详细信息、价格走势图、并提供买入/卖出操作入口。

---

## 📐 页面结构

### 1. 顶部统计区 (24小时数据卡片)
4个卡片，2x2 网格布局：

| 左上 | 右上 |
|------|------|
| **24小时最高** | **24小时最低** |
| $67866.60 | $66606.70 |

| 左下 | 右下 |
|------|------|
| **24小时量** | **24小时额** |
| 2,098.04 SOL | 1.40亿 |

**设计要求：**
- 白色圆角卡片背景
- 标签文字灰色 (#9ca3af)，14px
- 数值文字深色 (#1f2937)，20px，加粗
- 卡片间距 12px
- 整体卡片容器有轻微阴影

---

### 2. Tab 切换区
3个 Tab 选项：
- **图表**（默认选中，底部有下划线指示器）
- 最新成交
- 持币量

**设计要求：**
- Tab 文字颜色：未选中 #9ca3af，选中 #1f2937
- 下划线指示器：蓝色 (#667eea)，宽度 40px，高度 2px
- Tab 间距均匀分布
- 底部有浅灰色分隔线

---

### 3. 图表显示区
- **占位区域**：当前为空白图表，需要集成图表库（推荐 echarts 或 lightweight-charts）
- **右侧 Y 轴**：显示价格刻度（108.00, 110.00, 112.00, 114.00, 116.00）
- **图表类型**：K线图或折线图
- **交互要求**：支持缩放、拖拽、十字光标

---

### 4. 底部操作区
2个按钮，左右排列：

| 左侧 | 右侧 |
|------|------|
| **买入**（绿色按钮） | **卖出**（红色按钮） |

**按钮样式：**
- 高度：56px
- 圆角：12px
- 文字：白色，18px，加粗
- 买入背景色：#4CAF50（绿色）
- 卖出背景色：#F44336（红色）
- 按钮间距：16px
- 底部留白：适配 iOS 安全区

---

## 🎨 设计规范

### 颜色方案
- **背景色**：#f8f9fa（浅灰色）
- **卡片背景**：白色 (#ffffff)
- **主文字**：#1f2937（深灰）
- **次要文字**：#9ca3af（浅灰）
- **买入色**：#4CAF50（绿色）
- **卖出色**：#F44336（红色）

### 间距规范

- 卡片间距：12px
- 按钮间距：16px
- Tab 间距：均匀分布

### 圆角规范
- 卡片圆角：12px
- 按钮圆角：12px
- 图表容器圆角：12px

### 阴影
- 卡片阴影：0 2px 8px rgba(0, 0, 0, 0.08)
- 按钮阴影：0 4px 12px rgba(0, 0, 0, 0.15)

---

## 📱 响应式要求

### iOS 适配
- 顶部适配刘海屏：`padding-top: env(safe-area-inset-top)`
- 底部适配 Home 条：`padding-bottom: env(safe-area-inset-bottom)`
- 使用 `100dvh` 而非 `100vh`

### 横屏适配
- 最小宽度：320px
- 图表区域自动调整高度

---

## 🔧 技术实现建议

### 推荐技术栈
- **图表库**：echarts 或 lightweight-charts（TradingView）
- **数据源**：从 `marketStore` 获取实时价格数据
- **WebSocket 更新**：监听 `cryptoPriceUpdate` 事件实时更新图表

### 数据接口
```javascript
// 从 marketStore 获取数据
const marketStore = useMarketStore()
const coinData = computed(() => {
  return marketStore.marketData.find(c => c.symbol === route.params.symbol)
})

// WebSocket 实时更新
socket.on('cryptoPriceUpdate', (data) => {
  // 更新图表数据
  updateChartData(data)
})
```
